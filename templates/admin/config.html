Setup Guide

Custom SMTP

Enter your mail server hostname and port. Common settings:

  • Port 587 — STARTTLS (recommended)
  • Port 465 — SSL/TLS
  • Port 25 — No encryption

Gmail / Google Workspace

App Password required

  1. Enable 2-Step Verification on your Google account
  2. Go to Google Account → Security → App Passwords
  3. Generate a password for "Mail"
  4. Use that 16-character code as the password here

Connection options:

If port 587 fails with WinError 10061, try port 465.

Microsoft 365

Server auto-set to smtp.office365.com:587

Requirements

  1. SMTP AUTH must be enabled in your Microsoft 365 admin center
  2. Use your full email address as username
  3. Use your Microsoft 365 password (or App Password if MFA is on)

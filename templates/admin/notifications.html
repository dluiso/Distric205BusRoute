CSV Format Requirements

  • Required columns: group, first_name or email
  • Optional: household_label, role, last_name, phone
  • role must be parent or student (default: parent)
  • Rows with the same group + household_label are merged into one enrollment with multiple contacts
  • Groups must be created first — unknown group names are skipped
  • File must be UTF-8 encoded (Excel exports as UTF-8 with BOM work fine)
